# go-num2ja

num2ja provide ToKanji and ToHira for convertion between number and kanji/hiragana

## Usage

```
fmt.Println(num2ja.ToKanji(213)) // 二百十三
fmt.Println(num2ja.ToHira(213)) // にひゃくじゅうさん
```

## Installation

```
go get github.com/mattn/go-num2ja
```
